# SOWA E-Commerce Web Application – Digital Wall Art Shop

This project is a simple E-Commerce Web Application developed as an individual assignment for the Service Oriented Web Applications (SOWA) module.

The system demonstrates how a front-end web interface communicates with a back-end REST API built using Spring Boot. The application allows users to view available digital wall art products and perform basic shopping cart operations.

The project follows a Service-Oriented architecture where the front-end retrieves and updates product data through API endpoints provided by the back-end service.
